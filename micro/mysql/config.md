# MySQL配置

MySQL客户端的配置参数如下所示：

```php
[
    'host'      => '127.0.0.1',     // 主机名 必填
    'port'      => 3306,            // 端口号 选填，默认值3306
    'user'      => 'root',          // 用户名 必填
    'password'  => '123456',        // 密码  必填
    'database'  => 'Test',          // 数据库名称 必填
    'charset'   => 'utf8',          // 字符集 选填 默认值utf8  
]
```
